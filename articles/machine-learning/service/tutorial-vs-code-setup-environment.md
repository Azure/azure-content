# Learn how to set up Azure Machine Learning Visual Studio Code extension

## Prerequisites

- Visual Studio Code
- Python 3.5 or later

## Install the extensions

1. Open VS Code.
1. Select **Extensions** tab.
1. In the searchbox, enter "Azure Machine Learning".
1. Select **Install**.
